Browser Object Model
--------------------
* root object : `window`
    * It's an optional thing to mention
    * properties available in the window object can be directly called without doing window. before them
    * alert(), confirm(), prompt(), close(), open() are methods of the window object

Document Object Model
---------------------
--> DOM is a tree like structure of all the elements in the HTML document
    * DOM Parser: used to parse the HTML document and create a tree like structure of all the elements in the document
    * DOM API: used to interact with the DOM tree


backticks(`) : used to create a template string(holds multiple lines of code)
