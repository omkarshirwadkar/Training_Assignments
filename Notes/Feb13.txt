--> Paste the Password

To run a java file:
--> Add jdk home
--> Add maven home
--> Choose Execute Windows Batch Command and add the following script:
    --> cd C:\Users\Wissen\Desktop\Jenkins
        javac ABC.java
        set classpath=.;
        java ABC

--> Add parameters
